About this hub

The Splunk Learning Hub gathers the full Splunk Core User course — eight lessons and a hands-on lab portal — into one installable dashboard, so everything works from a single, searchable home.

Install it

On desktop Chrome or Edge, use Install app in the top bar. On iPhone, tap Share then Add to Home Screen. It then opens full-screen like a native app.

Track progress

Opening a lesson marks it in progress; use the check to mark it done. Your status and bookmarks are saved on this device only.

Works offline

Pages you have already opened stay available without a connection, so you can review on the go.

Search everything

Use the top search to jump to any lesson by title, topic, or concept. Use Web Search for anything beyond the course.
